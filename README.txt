#### Coyote 1.0 ####This application was developed completely in React.js and it makes use of Binance API, Plotly and Bootstrap 4. Please install all the required packages, listed in package.json file.####This app was developed in order to give an indication about the target sell price for achieving a fixed percentage of profit.  It takes into account the maker fee detracted at both the buy and sell actions. The graphical output, which the application gives to the user, gives also an inside whether the volatility of the market could provide the desired profit. ####The application is not meant to be in any case a financial advice.
